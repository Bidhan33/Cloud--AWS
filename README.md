## 1. AWS Certified Cloud Practitioner

### Overview:
The **AWS Certified Cloud Practitioner** certification is intended for individuals who are new to the cloud and wish to gain foundational knowledge of AWS services, security, and the AWS Cloud ecosystem. This certification is perfect for those who want to demonstrate their knowledge of AWS Cloud concepts at a high level.

### Key Topics:
- AWS Cloud concepts
- AWS core services and their use cases
- AWS pricing and billing models
- AWS Cloud security fundamentals
- AWS architecture best practices

### Exam Details:
- Duration: 90 minutes
- Format: Multiple Choice, Multiple Response
- Passing Score: 700 out of 1000

### Resources:
- [AWS Certified Cloud Practitioner Exam Guide](https://aws.amazon.com/certification/certified-cloud-practitioner/)
- [Cloud Practitioner Exam Questions](#cloud-practitioner-exam-questions)
